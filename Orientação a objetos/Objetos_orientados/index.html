Objeto orientado

Objetos

Objeto é tudo aquilo que apresenta propiedades, ou seja, características próprias.

Classes:

Cada objeto pode ter uma forma de ser executado, e essa forma é chamada de Classe, fazendo com que todos os objetos iguais tenham as mesmas propriedades e métodos.

Métodos:

Os métodos são o que fazem o objeto executar ações como exemplo falar ou dançar.

Criando objetos:

Cada objeto é uma versão específica da Classe, cada um com suas próprias propriedades e métodos.

Usando objetos:

Depois de criar um objeto, você poderá fazer com que ele realize ações,3 no casos métodos, e verificar as propriedades.

Basicamente programação orientada a objetos em JavaScript é criar objetos com classes (modelo) e fazer com que esses objetos tenham propriedades (características) e métodos (ações).

REGEX

O regEx ou expressões regulares é um método formal para especificar um padrão de texto, com essas expressões podemos definir como os dados devem estar formatados para serem considerados válidos


Esses símbolos como /\S+@\S+\.\S+/ misturados com caracteres normais, são chamados de metacaracteres, que formam uma expressão regular, que quando ela é testada com algum texto é possivel saber se essa expressão casa com ele ou não, fazendo com que seja possivel criar validações e maskaras, que são padrões que seriam difíceis de se fazer usando programação.


No card ao lado na parte da frente temos a imagem do código que foi aplicado para fazer o formulário da validação, já na parte de trás temos um exemplo de como funciona os caracteres do regEx.


Como exemplo podemos pegar a expressão regular que foi usada para a validação de um CPF no formulário: "/^\d{3}\.\d{3}\.\d{3}-\d{2}$/".

Essa expressão regular valida se a entrada fornecida no formulário como CPF segue o padrão de um CPF brasileiro, garantindo que os dígitos estejam nos lugares corretos e separados pelos caracteres específicos.

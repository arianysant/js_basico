JQUERY

O jQuery é uma biblioteca de JavaScript rápida, onde sua idéia é facilitar a vida dos desenvolvedores Web.


O jQuery simplifica tarefas como a criação de animações e a adição de funcões, tornando mais acessíveis para os desenvolvedores realizarem ações legais e interessantes em seus sites sem ter que escrever muito código JavaScript


Como seu próprio slogan já diz, "Escreva menos, faça mais".


Na imagem ao lado temos um exemplo de um código jQuery com JS puro, projetado para coletar dados de um formulário, possivelmente para realizar verificações ou validações no processo de envio.
